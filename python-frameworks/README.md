## Build Order (Recommended)

```
1. Django        → web apps (most jobs)
2. FastAPI       → modern APIs (always useful)
3. SQLAlchemy    → database work (in every project)
4. Pydantic      → validation (in FastAPI + everywhere)
5. pytest Deep   → testing (write reliable code)
6. NumPy/Pandas  → data analysis (if interested)
7. Celery        → async tasks (production necessity)
8. scikit-learn  → ML intro (if going data route)
9. PyTorch       → deep learning (after sklearn)
10. LangChain    → LLMs (after PyTorch)
11. Playwright   → browser testing (for web devs)
12. Airflow      → data pipelines (specialized)
13. Flask        → microservices (simple projects)
```
