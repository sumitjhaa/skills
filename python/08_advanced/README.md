## Advanced Nitigrities

These mini-files focus on **real-world library patterns** — each is a standalone, runnable recipe book for a standard library module you'll reach for daily in production.

| # | Module | File | What You'll Learn |
|---|--------|------|-------------------|
| 20 | contextlib | [08-20-contextlib-patterns.py](code/08-20-contextlib-patterns.py) | `@contextmanager` with try/finally, `ExitStack` for dynamic cleanup, `suppress`, `redirect_stdout`, `nullcontext` |
| 21 | functools | [08-21-functools-patterns.py](code/08-21-functools-patterns.py) | `@cached_property`, `@singledispatchmethod`, `@cache` (unbounded memoization), `partial` for config calls |
| 22 | itertools | [08-22-itertools-recipes.py](code/08-22-itertools-recipes.py) | `batched`, `pairwise`, `islice` pagination, `zip_longest`, `compress`, `chain.from_iterable` |
| 23 | typing | [08-23-typing-patterns.py](code/08-23-typing-patterns.py) | `NewType`, `Annotated`, `cast`, `TYPE_CHECKING`, `Final`, `Literal`, `NamedTuple` with defaults |
| 24 | pathlib | [08-24-pathlib-deep.py](code/08-24-pathlib-deep.py) | `rglob`, `with_suffix`, `stat`, `relative_to`, `resolve`, `write_text`, `mkdir(parents=True)` |
| 25 | re | [08-25-re-patterns.py](code/08-25-re-patterns.py) | `re.VERBOSE`, named groups, lookahead, `finditer`, span/start/end methods |
| 26 | json | [08-26-json-patterns.py](code/08-26-json-patterns.py) | `cls=CustomEncoder`, `object_hook` decoder, `ensure_ascii`, `indent`, `sort_keys` |
| 27 | logging | [08-27-logging-patterns.py](code/08-27-logging-patterns.py) | `dictConfig`, `RotatingFileHandler`, JSON format, `extra` context, `__name__` logger |
| 28 | csv | [08-28-csv-patterns.py](code/08-28-csv-patterns.py) | `DictReader`/`DictWriter`, `Sniffer` dialect detection, `QUOTE_ALL`, custom delimiter |
| 29 | gc & weakref | [08-29-gc-weakref-patterns.py](code/08-29-gc-weakref-patterns.py) | `WeakValueDictionary` observer, `gc.get_objects()`, `gc.collect()`, `weakref.finalize` |
| 30 | unittest.mock | [08-30-mock-patch-mock.py](code/08-30-mock-patch-mock.py) | `Mock(spec=)`, `PropertyMock`, `patch.dict`, `AsyncMock`, `MagicMock` |
