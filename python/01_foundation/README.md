# Phase 01: Foundation 🧱

Welcome to the first phase of your Python journey! This phase covers the absolute fundamentals — the building blocks you'll use in every Python program you ever write. From printing your first "Hello, World!" to understanding truthy and falsy values, you'll build a solid foundation that makes everything else easier.

Each lesson includes a concise markdown explanation, a runnable code file, and links to related topics. Work through them in order — later lessons build on earlier ones.
