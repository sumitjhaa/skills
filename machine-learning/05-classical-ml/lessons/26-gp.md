## Learning Objectives
- Understand GP as a distribution over functions
- Implement GP regression with kernel functions
- Optimize hyperparameters via marginal likelihood
- Apply approximations for scalability

## Model Definition
A Gaussian Process is a collection of random variables, any finite subset of which has a joint Gaussian distribution:

$$f(x) \sim \mathcal{GP}(m(x), K(x, x'))$$

- $m(x) = \mathbb{E}[f(x)]$: mean function (often set to 0)
- $K(x, x') = \text{Cov}(f(x), f(x'))$: covariance (kernel) function

The GP defines a prior over functions. After observing data, we compute the posterior over functions.

## Prediction
Joint distribution of training outputs $y$ and test outputs $f_*$:

$$\begin{bmatrix} y \\ f_* \end{bmatrix} \sim \mathcal{N}\left(0, \begin{bmatrix} K_{XX} + \sigma_n^2 I & K_{X*} \\ K_{*X} & K_{**} \end{bmatrix}\right)$$

### Posterior (Conditional)
$$\bar{f}_* = K_{*X} (K_{XX} + \sigma_n^2 I)^{-1} y$$

$$\text{Cov}(f_*) = K_{**} - K_{*X} (K_{XX} + \sigma_n^2 I)^{-1} K_{X*}$$

The predictive mean is a linear combination of observed $y$'s with weights given by kernel similarities.

### Noise-Free Case
When $\sigma_n^2 = 0$, the GP interpolates exactly: $\bar{f}_*(X) = y$, $\text{Cov}(f_*(X)) = 0$.

## Common Kernels

| Kernel | $K(x, x')$ | Properties | Length scale |
|--------|-----------|------------|--------------|
| RBF (Squared Exponential) | $\exp(-\frac{\|x-x'\|^2}{2l^2})$ | Infinitely smooth | $l$ |
| Matérn 1/2 | $\exp(-r/l)$ | Rough, once diff. | $l$ |
| Matérn 3/2 | $(1+\sqrt{3}r/l)\exp(-\sqrt{3}r/l)$ | Once diff. | $l$ |
| Matérn 5/2 | $(1+\sqrt{5}r/l+5r^2/3l^2)\exp(-\sqrt{5}r/l)$ | Twice diff. | $l$ |
| Periodic | $\exp(-\frac{2\sin^2(\pi|x-x'|/p)}{l^2})$ | Periodic | $p$ |
| Linear | $x^\top x'$ | Linear functions | — |
| Rational Quadratic | $(1 + r^2/(2\alpha l^2))^{-\alpha}$ | Multi-scale | $l, \alpha$ |

where $r = |x - x'|$.

## Hyperparameter Learning
Optimize log marginal likelihood:

$$\log p(y|X, \theta) = -\frac12 y^\top K_y^{-1} y - \frac12 \log|K_y| - \frac{n}{2} \log 2\pi$$

where $K_y = K_{XX} + \sigma_n^2 I$.

### Gradient w.r.t. Hyperparameters
$$\frac{\partial}{\partial \theta_j} \log p(y|X, \theta) = \frac12 \text{tr}\left((\alpha \alpha^\top - K_y^{-1}) \frac{\partial K_y}{\partial \theta_j}\right)$$

where $\alpha = K_y^{-1} y$.

The marginal likelihood automatically balances data fit (first term) vs model complexity (second term) — Occam's razor.

## Code: GP Regression

```python
import numpy as np
from scipy.linalg import cholesky, solve_triangular

class GaussianProcess:
    def __init__(self, kernel='rbf', length_scale=1.0, sigma_noise=0.1):
        self.length_scale = length_scale
        self.sigma_noise = sigma_noise

    def _kernel(self, X1, X2):
        sq_dist = np.sum(X1**2, axis=1, keepdims=True) + np.sum(X2**2, axis=1) - 2 * X1 @ X2.T
        return np.exp(-0.5 * sq_dist / self.length_scale**2)

    def fit(self, X, y):
        self.X_train = X
        K = self._kernel(X, X) + self.sigma_noise**2 * np.eye(len(X))
        self.L_ = cholesky(K, lower=True)
        self.alpha_ = solve_triangular(self.L_.T, solve_triangular(self.L_, y, lower=True))

    def predict(self, X, return_std=False):
        K_s = self._kernel(X, self.X_train)
        mu = K_s @ self.alpha_
        if return_std:
            v = solve_triangular(self.L_, K_s.T, lower=True)
            var = self._kernel(X, X) - v.T @ v
            return mu, np.sqrt(np.diag(var))
        return mu
```

## Practical Considerations
- **$O(n^3)$ training**: Cholesky decomposition — limits to ~10k points
- **$O(n^2)$ prediction**: Per test point — use sparse approximations for speed
- **Numerical stability**: Add jitter ($10^{-6} I$) to kernel matrix for Cholesky
- **Kernel choice**: RBF is default; Matérn more realistic for physical processes
- **Input warping**: Transform inputs before GP for non-stationary behavior
- **Multi-output**: Use coregionalization kernels (linear model of coregionalization)

### Scalable Approximations
- **Subset of Data**: Use $m \ll n$ points ($O(m^3)$)
- **Nyström approximation**: $K \approx Q_{mm} Q^{-1}_{mm} Q_{mm}^\top$
- **Inducing Points (FITC/VFE)**: $m$ pseudo-inputs, $O(nm^2)$
- **KISS-GP**: Structured kernel interpolation, $O(n + m^2)$
- **Scalable GPs**: SVGP (stochastic variational GP) for large-scale data

## Key Properties
- Full probabilistic predictions (mean + variance)
- Non-parametric — model complexity adapts to data
- Automatic Occam's razor via marginal likelihood
- $O(n^3)$ training limits to ~10k points without approximations
- Exact uncertainty quantification (unlike neural networks)
