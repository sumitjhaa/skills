## How to Use These

**When you're in a crisis:**
→ Open `grounding-45s.md`. Do the protocol. Then find the relevant doc.

**When you're stuck in a pattern:**
→ Open the doc for that pattern. Read the "evolutionary purpose" section. Then read the "how to counter" section.

**When you want to build a system:**
→ Start with `habits-science.md` + `motivation.md`. Build the foundation. Then layer on `focus-flow.md`.
