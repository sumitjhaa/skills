# 🖥️ IT Support → SysAdmin — Baby Style (6 weeks)

**For:** Zero experience. Like fixing things. Want to work with computers.

**Time:** 3 hrs/day | **Goal:** IT Support Specialist → Junior SysAdmin

**Pay:** $40-55k (support) → $60-75k (sysadmin)

---

## What is this?

- IT Support = help people with computer problems (passwords, wifi, printers)
- SysAdmin = manage servers, networks, users
- Most tech careers start here. Very easy to get first job.
- After 1-2 years, move to DevOps, cybersecurity, or cloud.

---

## 🗓️ Week 1 — Computer Basics

### Day 1: Hardware
- CPU, RAM,硬盘, motherboard, power supply
- What each part does

**Do:** Open your computer (or watch a video). Identify each part.

### Day 2: Operating systems
- Windows vs macOS vs Linux
- Desktop, file explorer, task manager, control panel

**Do:** Open Task Manager. See running processes. End a task.

### Day 3: Networking basics
- IP address = your computer's address on the internet
- DNS = converts google.com to IP address
- Router vs modem vs switch

**Do:** Find your IP address (ipconfig on Windows, ifconfig on Mac/Linux).

### Day 4: IP + ports
- 192.168.x.x = private IP (inside your house)
- Ports = doors on your computer (80 = web, 443 = secure web, 22 = SSH)

**Do:** Check which ports are open on your computer.

### Day 5: Troubleshooting
- "Is it plugged in?" → "Restart it" → "Check cables" → "Google the error"
- 90% of IT support is: restart, check cables, Google, ask user to try again

**Do:** Practice troubleshooting: intentionally break something, then fix it.

### Day 6: Customer service
- IT support is 50% people, 50% tech
- Stay calm. Ask questions. Write down what they say. Repeat back to them.

**Do:** Explain a technical thing to a non-technical person. Did they understand?

### Day 7: Practice
- Set up a home network. Configure router. Connect devices. Troubleshoot a "no internet" issue.

---

## 🗓️ Week 2 — Windows + Active Directory

### Day 1: Windows administration
- User accounts: create, delete, change passwords
- Groups: add users to groups
- Permissions: read/write/execute for files and folders

**Do:** Create 3 user accounts on your computer. Give them different permissions.

### Day 2: Active Directory (AD)
- AD = central directory of all users in a company
- Used by 90% of companies
- Users, computers, groups, organizational units (OUs)

**Do:** Set up a free Windows Server VM. Create users in AD.

### Day 3: Group Policy
- GPO = enforce settings on all computers (set wallpaper, block USB, etc.)

**Do:** Create a GPO that sets homepage to a specific URL.

### Day 4: Remote desktop
- RDP = control another computer remotely
- TeamViewer, AnyDesk

**Do:** Set up Remote Desktop. Connect from one computer to another.

### Day 5: Ticketing systems
- Jira, ServiceNow, Zendesk
- Users submit tickets. You resolve them. Track everything.

**Do:** Create tickets in a free Jira account. Assign, comment, resolve.

### Day 6-7: Practice
- Set up a small office network (virtual): router, 3 computers, printer, shared folder

---

## 🗓️ Week 3 — Linux Basics

### Day 1: Linux for sysadmins
- `ls`, `cd`, `pwd`, `mkdir`, `rm`, `cp`, `mv`
- `cat`, `less`, `tail`, `head`, `grep`

**Do:** Navigate the Linux file system. Find log files.

### Day 2: Users + permissions
- `useradd`, `passwd`, `usermod`, `groupadd`
- `chmod`, `chown`
- `sudo`

**Do:** Create users. Set passwords. Give sudo access.

### Day 3: Processes + services
- `systemctl start/stop/restart nginx`
- `journalctl` = view logs

**Do:** Install nginx. Start it. Make it serve a webpage. Check logs.

### Day 4: Bash scripting
- `#!/bin/bash`, variables, if/else, loops
- Write scripts to automate tasks

**Do:** Write a script that backs up a folder every day (copy with date in filename).

### Day 5: SSH
- `ssh user@ip` — connect to remote server
- SSH keys for password-less login

**Do:** Set up SSH keys. Connect to a server without typing password.

### Day 6-7: Linux project
- Set up a Linux server with: nginx, user accounts, firewall, automatic backups

---

## 🗓️ Week 4 — Networking for SysAdmins

### Day 1: TCP/IP + OSI model
- How data travels across the internet
- Layers: physical → network → transport → application

**Do:** Use `traceroute` to see how data travels to google.com.

### Day 2: DHCP + DNS
- DHCP = automatically assigns IP addresses
- DNS = resolves domain names to IPs

**Do:** Set up a DHCP server. Set up a DNS server.

### Day 3: Firewalls
- Block unwanted traffic
- Allow only specific ports

**Do:** Set up a firewall. Block all ports except 80 and 443.

### Day 4: VPN
- VPN = secure tunnel to company network
- WireGuard or OpenVPN basics

**Do:** Set up a WireGuard VPN. Connect from another device.

### Day 5: Monitoring
- Ping = is server alive?
- Uptime monitoring = alert if server goes down

**Do:** Set up a simple monitoring script that emails you when a server is down.

### Day 6-7: Networking project
- Set up a network with: router, DHCP, DNS, firewall, VPN, monitoring

---

## 🗓️ Week 5 — Certifications + Resume

### Day 1: CompTIA A+
- Entry level cert. Covers hardware, networking, troubleshooting.
- Costs ~$250. Helps get first IT job.

**Do:** Watch Professor Messer videos on YouTube (free). Take practice tests.

### Day 2: CompTIA Network+
- If you want to go deeper. Covers networking fully.

### Day 3: Resume
- 1 page. "IT Support Specialist"
- Skills: Windows, Linux, Active Directory, networking, troubleshooting, customer service

### Day 4: LinkedIn
- Headline: "IT Support Specialist"
- Skills section: add all the tech you know

### Day 5-6: Apply
- Roles: IT Support Specialist, Help Desk Technician, Desktop Support, Junior SysAdmin
- Indeed, LinkedIn — 10 apps/day
- Entry level IT support is always hiring

### Day 7: Interview prep
- "Tell me about a time you fixed a technical problem"
- "How do you handle an angry user?"
- "What's the difference between TCP and UDP?"

---

## 🗓️ Week 6 — Cert exam + Keep applying

- Take CompTIA A+ exam (if ready)
- Keep applying 10/day
- After 1-2 years in IT support → move to SysAdmin ($60-75k) or DevOps ($80-100k)
