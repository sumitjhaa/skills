## Mathematical Foundation

### Probability Space

A **probability space** is a triple $(\Omega, \mathcal{F}, P)$ where:

- **$\Omega$** is the sample space — the set of all possible outcomes of a random experiment
- **$\mathcal{F}$** is a sigma-algebra (or $\sigma$-algebra) — a collection of events (subsets of $\Omega$)
- **$P$** is a probability measure — a function $P: \mathcal{F} \to [0,1]$ that assigns probabilities to events

### Sigma-Algebra Axioms

A collection $\mathcal{F}$ of subsets of $\Omega$ is a $\sigma$-algebra if:

1. **Contains the sample space:** $\Omega \in \mathcal{F}$
2. **Closed under complement:** If $A \in \mathcal{F}$, then $A^c = \Omega \setminus A \in \mathcal{F}$
3. **Closed under countable unions:** If $A_1, A_2, A_3, \dots \in \mathcal{F}$, then $\bigcup_{i=1}^{\infty} A_i \in \mathcal{F}$

From these, we also get closure under countable intersections (by De Morgan's laws):
$$\bigcap_{i=1}^{\infty} A_i = \left(\bigcup_{i=1}^{\infty} A_i^c\right)^c \in \mathcal{F}$$

**Why sigma-algebras?** In uncountable spaces (e.g., $\Omega = \mathbb{R}$), not all subsets are measurable. The $\sigma$-algebra restricts us to "well-behaved" sets where probability is defined consistently. This avoids paradoxes like the Banach-Tarski paradox.

### Probability Measure Axioms (Kolmogorov Axioms)

A function $P: \mathcal{F} \to [0,1]$ is a probability measure if:

1. **Non-negativity:** $P(A) \geq 0$ for all $A \in \mathcal{F}$
2. **Unit normalization:** $P(\Omega) = 1$
3. **Countable additivity ($\sigma$-additivity):** For any countable collection of disjoint events $A_1, A_2, \dots \in \mathcal{F}$ with $A_i \cap A_j = \emptyset$ for $i \neq j$:
   $$P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$$

## Key Properties Derived from the Axioms

### Elementary Consequences

1. **Probability of the empty set:**
   $$P(\emptyset) = 0$$
   _Proof:_ $\Omega = \Omega \cup \emptyset \cup \emptyset \cup \cdots$, so $P(\Omega) = P(\Omega) + P(\emptyset) + P(\emptyset) + \cdots$, implying $P(\emptyset) = 0$.

2. **Probability of complement:**
   $$P(A^c) = 1 - P(A)$$
   _Proof:_ $\Omega = A \cup A^c$, disjoint, so $1 = P(A) + P(A^c)$.

3. **Finite additivity:**
   $$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$
   _Proof:_ $A \cup B = A \cup (B \setminus A)$, disjoint, and $B = (A \cap B) \cup (B \setminus A)$, disjoint.

4. **Monotonicity:** If $A \subseteq B$, then $P(A) \leq P(B)$

5. **Subadditivity (Boole's inequality):**
   $$P\left(\bigcup_{i=1}^{\infty} A_i\right) \leq \sum_{i=1}^{\infty} P(A_i)$$

### Continuity of Probability

The probability measure is **continuous from below**: If $A_1 \subseteq A_2 \subseteq A_3 \subseteq \dots$ and $A = \bigcup_{n=1}^{\infty} A_n$, then:
$$\lim_{n \to \infty} P(A_n) = P(A)$$

**Continuous from above**: If $A_1 \supseteq A_2 \supseteq A_3 \supseteq \dots$ and $A = \bigcap_{n=1}^{\infty} A_n$, then:
$$\lim_{n \to \infty} P(A_n) = P(A)$$

## Borel Sigma-Algebra

The **Borel sigma-algebra** $\mathcal{B}(\mathbb{R})$ is the smallest $\sigma$-algebra on $\mathbb{R}$ that contains all open intervals. It is generated by any of the following:
- All open intervals $(a, b)$
- All closed intervals $[a, b]$
- All half-open intervals $(a, b]$ or $[a, b)$
- All open sets
- All half-lines $(-\infty, a)$

$\mathcal{B}(\mathbb{R})$ contains virtually every set encountered in practice — all intervals, open sets, closed sets, countable unions/intersections of these. However, it does not contain all subsets of $\mathbb{R}$ (non-measurable sets exist).

## Examples of Probability Spaces

### Example 1: Fair Coin Toss (Discrete)

$$\Omega = \{H, T\}, \quad \mathcal{F} = \{\emptyset, \{H\}, \{T\}, \{H, T\}\}, \quad P(H) = P(T) = 0.5$$

All subsets are measurable. This is the simplest non-trivial probability space.

### Example 2: Uniform Distribution on [0,1] (Continuous)

$$\Omega = [0,1], \quad \mathcal{F} = \mathcal{B}([0,1]), \quad P([a,b]) = b - a \text{ for } 0 \leq a \leq b \leq 1$$

This is the **Lebesgue measure** restricted to [0,1]. It models "uniformly random" choice from the unit interval.

### Example 3: Countably Infinite (Geometric)

$$\Omega = \mathbb{N} = \{1, 2, 3, \dots\}, \quad \mathcal{F} = 2^{\Omega} \text{ (all subsets)}, \quad P(\{k\}) = (1-p)^{k-1}p$$

### Example 4: Product Space (Multiple Independent Experiments)

For independent coin flips, $\Omega = \{H, T\}^n$, $\mathcal{F} = 2^{\Omega}$, and:
$$P(\{\omega_1, \dots, \omega_n\}) = \prod_{i=1}^n P(\{\omega_i\})$$

## Product Sigma-Algebra

For a product space $\Omega = \Omega_1 \times \Omega_2$, the **product sigma-algebra** $\mathcal{F}_1 \otimes \mathcal{F}_2$ is the smallest $\sigma$-algebra containing all rectangles $A_1 \times A_2$ for $A_1 \in \mathcal{F}_1, A_2 \in \mathcal{F}_2$. This is essential for defining joint distributions of multiple random variables.

## Python Implementation

```python
import numpy as np

class ProbabilitySpace:
    """A simple discrete probability space."""
    def __init__(self, outcomes, probabilities):
        self.outcomes = list(outcomes)
        self.probs = np.array(probabilities)
        assert len(self.outcomes) == len(self.probs)
        assert np.isclose(self.probs.sum(), 1.0)
        assert (self.probs >= 0).all()

    def probability(self, event_mask):
        """P(event) where event_mask is a boolean array over outcomes."""
        return self.probs[event_mask].sum()

    def complement(self, event_mask):
        return 1.0 - self.probability(event_mask)

    def union(self, mask_a, mask_b):
        pa = self.probability(mask_a)
        pb = self.probability(mask_b)
        pab = self.probability(mask_a & mask_b)
        return pa + pb - pab

# Example: fair die
die = ProbabilitySpace([1, 2, 3, 4, 5, 6], [1/6]*6)
even = np.array([False, True, False, True, False, True])
odd = ~even
print(f"P(even) = {die.probability(even)}")
print(f"P(odd) = {die.probability(odd)}")
print(f"P(even ∪ odd) = {die.union(even, odd)}")
```

## Visualization

A Venn diagram of the probability space helps visualize the axioms. Plot overlapping events $A$ and $B$ with areas proportional to probabilities. The union formula $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ is directly visible as the total covered area. For disjoint events, the areas do not overlap, confirming $P(A \cup B) = P(A) + P(B)$.

A second plot showing nested events $A_1 \subset A_2 \subset \dots$ illustrates continuity from below: the probability of the limiting set is the limit of the probabilities.

## Practical Considerations

- **Measurability matters:** In most ML applications, you work with Borel sets on $\mathbb{R}^d$ and never worry about non-measurable sets. However, when defining stochastic processes or Bayesian nonparametrics, measure-theoretic foundations become important.
- **Discrete vs continuous:** Discrete probability spaces use $\mathcal{F} = 2^{\Omega}$ (all subsets). Continuous spaces require Borel $\sigma$-algebras. Many practical algorithms (e.g., Monte Carlo) discretize continuous problems.
- **Countable additivity vs finite additivity:** Some philosophers (de Finetti) argue for finite additivity only, but countable additivity is essential for limit theorems and integration theory used throughout ML.
- **Sigma-algebra generation:** When working with random variables, the $\sigma$-algebra generated by the variable (the smallest $\sigma$-algebra making it measurable) captures all information obtainable from that variable.
