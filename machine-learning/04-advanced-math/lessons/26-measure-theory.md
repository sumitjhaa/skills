# 04.26 Measure Theory and the Lebesgue Integral

## Motivation
Measure theory provides a rigorous foundation for probability, expectation, and integration. It is essential for understanding convergence of random variables, statistical models, and the theoretical underpinnings of modern ML — from density estimation and variational inference to generative modelling and domain generalisation.

## Learning Objectives
- Define $\sigma$-algebras, measures, and measurable functions.
- Construct the Lebesgue integral and state the key convergence theorems.
- Understand Radon–Nikodym derivatives and their role in probability.
- Apply measure-theoretic concepts to ML: densities under different dominating measures, GAN theory, and distribution shift.

## Math Foundation

### $\sigma$-Algebras and Measurable Spaces
A $\sigma$-algebra $\mathcal{F}$ on a set $\Omega$ is a collection of subsets closed under:
1. $\Omega \in \mathcal{F}$.
2. $A \in \mathcal{F} \implies A^c \in \mathcal{F}$.
3. $A_i \in \mathcal{F}$ for $i = 1,2,\dots \implies \bigcup_i A_i \in \mathcal{F}$.

The pair $(\Omega, \mathcal{F})$ is a measurable space. The Borel $\sigma$-algebra on $\mathbb{R}^d$ (denoted $\mathcal{B}(\mathbb{R}^d)$) is generated by open sets — it contains all intervals, open sets, closed sets, and their countable combinations.

### Measures
A measure $\mu: \mathcal{F} \to [0, \infty]$ is countably additive:

$$\mu\!\left(\bigsqcup_{i=1}^\infty A_i\right) = \sum_{i=1}^\infty \mu(A_i)$$

with $\mu(\emptyset) = 0$. A probability measure satisfies $\mu(\Omega) = 1$.

**Examples**:
- **Lebesgue measure $\lambda$**: length on $\mathbb{R}$, area on $\mathbb{R}^2$, $d$-dimensional volume.
- **Dirac measure**: $\delta_x(A) = 1$ if $x \in A$, 0 otherwise.
- **Counting measure**: $\nu(A) = |A|$ (cardinality).
- **Gaussian measure**: $\mu(A) = \int_A \frac{1}{\sqrt{2\pi\sigma^2}} e^{-(x-m)^2/2\sigma^2} dx$.

### Measurable Functions
A function $f: \Omega \to \mathbb{R}$ is measurable if $f^{-1}(B) \in \mathcal{F}$ for all $B \in \mathcal{B}(\mathbb{R})$. Continuous functions, indicator functions, and their pointwise limits are all measurable.

### Lebesgue Integral
The Lebesgue integral is constructed in three steps:
1. **Simple functions**: $f = \sum_{i=1}^n a_i \mathbf{1}_{A_i}$, define $\int f d\mu = \sum a_i \mu(A_i)$.
2. **Non-negative functions**: $\int f d\mu = \sup\{\int s d\mu : 0 \le s \le f, s \text{ simple}\}$.
3. **General functions**: $f = f^+ - f^-$ where $f^+ = \max(f,0)$, $f^- = \max(-f,0)$, define $\int f d\mu = \int f^+ d\mu - \int f^- d\mu$ provided at least one is finite.

### Convergence Theorems
1. **Monotone Convergence Theorem**: if $0 \le f_n \nearrow f$ a.e., then $\int f_n \to \int f$.
2. **Fatou's Lemma**: $\int \liminf f_n \le \liminf \int f_n$ for $f_n \ge 0$.
3. **Lebesgue Dominated Convergence Theorem**: if $f_n \to f$ a.e. and $|f_n| \le g$ with $\int g < \infty$, then $\int f_n \to \int f$.

These theorems justify interchanging limits and integrals — a luxury not available for the Riemann integral.

### Radon–Nikodym Theorem
If $\nu$ and $\mu$ are $\sigma$-finite measures with $\nu \ll \mu$ ($\nu$ is absolutely continuous w.r.t. $\mu$, meaning $\mu(A) = 0 \implies \nu(A) = 0$), then there exists a measurable function $f = d\nu/d\mu$ such that for all measurable $A$:

$$\nu(A) = \int_A f d\mu$$

The function $f$ is the Radon–Nikodym derivative (density). In probability, if $P \ll \lambda$, then $dP/d\lambda = p(x)$ is the probability density function.

### Product Measures and Fubini's Theorem
For complete measure spaces $(\Omega_1, \mathcal{F}_1, \mu_1)$ and $(\Omega_2, \mathcal{F}_2, \mu_2)$, there exists a unique product measure $\mu_1 \otimes \mu_2$ on $\mathcal{F}_1 \otimes \mathcal{F}_2$ such that:

$$(\mu_1 \otimes \mu_2)(A \times B) = \mu_1(A) \mu_2(B)$$

Fubini's theorem: if $f$ is integrable, then:

$$\int_{\Omega_1 \times \Omega_2} f d(\mu_1 \otimes \mu_2) = \int_{\Omega_1} \left( \int_{\Omega_2} f(x,y) \mu_2(dy) \right) \mu_1(dx) = \int_{\Omega_2} \left( \int_{\Omega_1} f(x,y) \mu_1(dx) \right) \mu_2(dy)$$

def radon_nikodym_example():
    """Approximate RN derivative from samples."""
    # True density: p(x) = 0.3 * N(-1, 0.5) + 0.7 * N(2, 0.8)
    np.random.seed(42)
    n = 10000
    mixture = np.random.rand(n) < 0.3
    samples = np.where(mixture,
                       np.random.randn(n) * 0.5 - 1,
                       np.random.randn(n) * 0.8 + 2)
    
    # Estimate density via histogram (approximate RN derivative w.r.t. Lebesgue)
    hist, edges = np.histogram(samples, bins=50, density=True)
    bin_centers = (edges[:-1] + edges[1:]) / 2
    
    # True density
    def true_density(x):
        return 0.3 * np.exp(-0.5 * ((x + 1) / 0.5)**2) / (0.5 * np.sqrt(2*np.pi)) \
             + 0.7 * np.exp(-0.5 * ((x - 2) / 0.8)**2) / (0.8 * np.sqrt(2*np.pi))
    
    true_vals = true_density(bin_centers)
    error = np.mean((hist - true_vals)**2)
    print(f"Histogram (plug-in) density estimation MSE: {error:.6f}")
    return bin_centers, hist, true_vals

def monte_carlo_integration(f, n_samples=10000):
    """Estimate Lebesgue integral via Monte Carlo."""
    samples = np.random.uniform(-1, 1, n_samples)
    integral = 2.0 * np.mean(f(samples))  # E[f] * volume
    return integral

# Example: compute E[sin(X)] for X ~ U(-1, 1)

## Connections to Machine Learning

### Probabilistic ML as Measure Theory
All of probabilistic ML is measure-theoretic:
- A statistical model is a family of probability measures $\{P_\theta : \theta \in \Theta\}$ on the data space.
- A likelihood is a Radon–Nikodym derivative w.r.t. a dominating measure.
- The posterior is $dP(\theta|x) = p(x|\theta) dP(\theta) / \int p(x|\theta) dP(\theta)$.
- Model comparison uses Bayes factors (ratios of marginal likelihoods), which are Radon–Nikodym derivatives.

### GANs and Pushforward Measures
A generator $G: \mathcal{Z} \to \mathcal{X}$ defines a pushforward measure $G_\# P_Z$:

$$(G_\# P_Z)(A) = P_Z(G^{-1}(A))$$

The GAN objective minimises a divergence between the pushforward and the data distribution:

$$\min_G D(P_{\text{data}} \| G_\# P_Z)$$

This is naturally expressed in measure-theoretic language: the optimal generator is the one that transforms the latent measure into the data measure.

### Domain Generalisation and Distribution Shift
Domain generalisation considers a family of related measures $\{P_e : e \in \mathcal{E}\}$ (environments). The goal is to learn a predictor that performs well across all environments. The invariant risk minimisation principle seeks a representation $\Phi$ such that $\mathbb{E}[Y | \Phi(X)]$ is the same measure for all $e$:

$$\mathbb{E}_{P_e}[Y | \Phi(X)] = \mathbb{E}_{P_{e'}}[Y | \Phi(X)] \quad \forall e, e'$$

This is a measure-theoretic conditional independence constraint.

### Optimal Transport and Measure Couplings
The Kantorovich formulation of optimal transport minimises:

$$W(P,Q) = \inf_{\gamma \in \Gamma(P,Q)} \mathbb{E}_{(X,Y) \sim \gamma}[c(X,Y)]$$

where $\Gamma(P,Q)$ is the set of couplings (probability measures on $\mathcal{X} \times \mathcal{Y}$ with marginals $P$ and $Q$). The existence of optimal couplings is guaranteed under mild conditions by Prokhorov's theorem and Skorokhod's representation.

## Practical Considerations

### Domination and Absolute Continuity
- Not all distributions have densities w.r.t. Lebesgue measure: discrete distributions are singular w.r.t. $\lambda$, and the uniform distribution on a fractal is singular continuous.
- In practice, always specify the dominating measure: counting measure for discrete, Lebesgue for continuous.
- The Radon–Nikodym theorem guarantees densities exist whenever $P \ll \mu$ — essential for likelihood-based inference.

### Convergence of Random Variables
Four modes of convergence (in increasing strength):
1. **In distribution**: $F_n \to F$ pointwise.
2. **In probability**: $P(|X_n - X| > \epsilon) \to 0$.
3. **Almost surely**: $P(\lim X_n = X) = 1$.
4. **In $L^p$**: $\mathbb{E}[|X_n - X|^p] \to 0$.

These are measure-theoretic notions that determine which ML convergence guarantees hold.
