# Python Mastery Course — Zero to Production

## What This Course Is

A complete, no-gaps Python curriculum designed for **ADHD/OCD minds** who need:
- **Small wins** — each lesson teaches exactly one thing
- **Predictable structure** — every phase follows the same pattern
- **Running code** — every concept has a file you can run immediately
- **Real connections** — integration files show how concepts work together
- **No fluff** — clean code files, documentation in markdown

## The Pattern

Every phase follows this structure:

```
phase/
├── README.md              # Phase overview, linked table of contents
├── lessons/
│   ├── 01-concept.md      # Documentation with examples
│   ├── 02-concept.md
│   └── ...                # Links to code files and related lessons
├── code/
│   ├── 01-concept.py      # Clean, runnable code (minimal comments)
│   ├── 02-concept.py
│   ├── ...
│   └── integration-theme.py  # Real-world integration example
└── practice/
    ├── exercises.md        # Problems with hints
    └── solutions/          # Working solutions
```

## How To Use This Course

1. **Read** the lesson markdown file — understand the concept
2. **Run** the corresponding code file — see it in action
3. **Study** the integration file — see concepts working together
4. **Do** the practice exercises — reinforce what you learned
5. **Move on** — each phase builds on the previous
