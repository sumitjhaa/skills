# 🤖 scikit-learn — Classical Machine Learning

30 lessons covering the scikit-learn ecosystem: supervised & unsupervised learning, preprocessing, model selection, pipelines, and production workflows.

## Structure

3 phases, 10 lessons each.

| Phase | Focus | Lessons | Practice |
|-------|-------|---------|----------|
| 01 | Classification & Clustering | 01–10 | ✅ |
| 02 | Modeling Workflow | 11–20 | ✅ |
| 03 | Advanced Topics | 21–30 | ✅ |

## How to Use

```bash
# Run a lesson's code
python code/01-train-test-split.py

# Read the corresponding lesson
lessons/01-train-test-split.md

# Do the practice exercises
practice/phase01-exercises.md
```

## Topics

**Phase 01:** train/test split, linear & logistic regression, decision trees, random forests, SVM, KNN, K-Means, PCA, classification integration.

**Phase 02:** feature engineering, feature selection, cross-validation, hyperparameter tuning, ensemble methods, gradient boosting, pipelines, model evaluation, imbalanced data, regression integration.

**Phase 03:** polynomial regression, ridge/lasso, naive Bayes, advanced clustering, model calibration, learning curves, save/load models, custom transformers, stacking/blending, full pipeline integration.

All code files run standalone with `python code/<file>.py` — no external data required.
