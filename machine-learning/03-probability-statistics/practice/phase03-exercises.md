# Phase 03 — Probability & Statistics: Practice Exercises

## Exercise 1: Sigma-Algebra Construction
Let Ω = {1, 2, 3, 4, 5, 6}. List all sets in the sigma-algebra generated by the partition {{1, 3, 5}, {2, 4, 6}}.

## Exercise 2: Transformation of Random Variables
If X ~ Uniform(0, 1), derive the PDF of Y = -ln(X). What family does Y belong to?

## Exercise 3: Central Limit Theorem Simulation
Write code to draw 10,000 samples of size n ∈ {5, 30, 100} from an exponential(1) distribution. Plot the histogram of sample means against the normal density predicted by the CLT.

## Exercise 4: MLE for a Gamma Distribution
Derive the MLE for the rate parameter λ of Gamma(α, λ) when α is known. Implement a numerical optimizer for both α and λ.

## Exercise 5: Bayesian Conjugate Analysis
For Binomial data with Beta prior, derive the posterior. Plot prior, likelihood, posterior for n = 20, y = 7, Beta(2, 2) prior. Compute 95% credible interval.

## Exercise 6: Hypothesis Testing Power Curve
Generate data from N(0.5, 1) and test H₀: μ = 0 at α = 0.05. Repeat 1000 times to estimate power. Vary effect size from 0 to 2 and plot the power curve.

## Exercise 7: Bootstrap Confidence Intervals
Given a sample of size 50 from a skewed distribution (e.g., log-normal), compute 95% confidence intervals for the median using: (a) normal approximation, (b) percentile bootstrap, (c) BCa bootstrap. Compare widths.

## Exercise 8: ANOVA Decomposition
Generate data from three groups with means 0, 1, 2. Compute the total sum of squares, between-group SS, and within-group SS. Verify SST = SSB + SSW. Run an F-test.

## Exercise 9: Logistic Regression via GLM
Simulate binary data with two predictors. Fit a logistic regression model using IRLS. Compare coefficients to statsmodels logit output.

## Exercise 10: Gaussian Process Regression
Generate 20 noisy observations from f(x) = sin(x). Fit a GP with RBF kernel. Plot the posterior mean and 95% credible band at 100 test points.

## Exercise 11: ARIMA Model Identification
Given the `AirPassengers` dataset, identify the correct ARIMA(p,d,q) order using ACF/PACF plots. Fit the model and forecast 12 months ahead with prediction intervals.

## Exercise 12: Kaplan-Meier Estimation
Simulate survival times from a Weibull distribution with censoring. Compute the Kaplan-Meier estimate and compare to the true survival function. Perform a log-rank test comparing two groups.

## Exercise 13: EM Algorithm for Gaussian Mixtures
Generate 200 points from a 2-component Gaussian mixture. Implement the EM algorithm from scratch to estimate means, variances, and mixing proportions. Plot the log-likelihood trajectory.

## Exercise 14: Causal Inference with DAGs
Draw a DAG for confounding. Simulate data where Z confounds X → Y. Compute the naive association and the causal effect using: (a) stratification, (b) inverse probability weighting, (c) regression adjustment.

## Exercise 15: Bayesian Workflow — Complete Example
Full workflow: (1) prior predictive simulation for a normal model, (2) fit with MCMC (use PyMC or manual Metropolis-Hastings), (3) posterior predictive checks, (4) compare two models via WAIC/LOO, (5) sensitivity analysis with different priors.

## Exercise 16: Empirical Process Simulation
Simulate n = 100 from N(0, 1). Plot the ECDF with 95% DKW confidence bands. Estimate the Kolmogorov-Smirnov test statistic and compare to the asymptotic distribution.

## Exercise 17: Copula Dependence Modeling
Generate 500 samples from a bivariate distribution with: (a) Gaussian copula ρ = 0.7, (b) Clayton copula θ = 2. Estimate Kendall's tau for each and verify the relationship with copula parameters.
