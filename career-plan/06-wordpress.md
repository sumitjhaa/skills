# 🌐 WordPress / Shopify — Baby Style (3-4 weeks)

**For:** No coding experience. Want to earn money FAST. Don't want to learn complex stuff.

**Time:** 2-3 hrs/day | **Goal:** Freelance web developer

**Pay:** $30-60/hr (freelance) | **Difficulty:** Easiest path to money

## What is this?

- WordPress = build websites without coding (60% of all websites use it)
- Shopify = build online stores
- Local businesses need websites. You build them. They pay you.
- You can make your first $500 in 2-3 weeks.

---

## 🗓️ Week 1 — WordPress Basics

### Day 1: What is WordPress?
- Login page → Dashboard → everything lives there
- Themes = design/layout
- Plugins = features (forms, galleries, SEO)
- Posts vs Pages: Posts = blog (dated). Pages = static (About, Contact).

**Do:** Install WordPress on your computer (use Local app — it's free and easy).

### Day 2: Dashboard tour
- Posts: write blog posts. Add categories, tags.
- Pages: create static pages (Home, About, Contact, Services)
- Media: upload images, videos

**Do:** Create a Home page, About page, and Contact page. Add some text.

### Day 3: Themes
- Themes change the entire look
- Free themes in Appearance → Themes
- Install, activate, customize

**Do:** Install a free theme. Customize the colors, logo, and layout.

### Day 4: Plugins
- Plugins add features
- Contact Form 7 or WPForms — add contact forms
- Yoast SEO — make site show up in Google
- Elementor — build pages visually (drag and drop)

**Do:** Install Contact Form 7. Add a form to your Contact page.

### Day 5: Customizing with Elementor
- Elementor = build pages like PowerPoint
- Drag text, images, buttons, columns
- No coding needed

**Do:** Rebuild your Home page with Elementor. Make it look professional.

### Day 6-7: Build a business site
**Do:** Build a complete site for a fake business (e.g., "Joe's Pizza"):
- Home page (hero image, about, menu preview)
- About page (story + team)
- Contact page (form + map + phone)
- Install SEO plugin. Set meta titles/descriptions.

---

## 🗓️ Week 2 — WooCommerce + Making Money

### Day 1: What is WooCommerce?
- WooCommerce = turns WordPress into an online store
- Add products, categories, cart, checkout, payments

**Do:** Install WooCommerce. Go through setup wizard.

### Day 2: Products
- Simple product = one item with one price
- Variable product = t-shirt in multiple sizes/colors
- Digital product = ebook, software, course

**Do:** Add 5 products with images, prices, and descriptions.

### Day 3: Store setup
- Payment: Stripe or PayPal
- Shipping: flat rate, free shipping, local pickup
- Tax settings

**Do:** Configure payment and shipping. Test checkout with a $1 product.

### Day 4: Shopify basics (bonus skill)
- Shopify = alternative to WooCommerce (hosted, simpler)
- Shopify has its own theme language (Liquid)
- You can build Shopify stores for clients

**Do:** Start a 14-day free Shopify trial. Add products. Customize theme.

### Day 5: PHP basics (for customization)
- Just enough PHP to edit WordPress themes
- `<?php echo "Hello"; ?>` — PHP tag
- `the_title()`, `the_content()`, `the_permalink()` — WordPress functions

**Do:** Open a theme file (header.php). Change the site title HTML. See it update.

### Day 6-7: Build an e-commerce site
**Do:** Full store for a fake brand:
- Products with categories
- Working cart + checkout
- Contact page
- About page

## 🗓️ Week 3 — Get Clients + Freelance

### Day 1: Portfolio
- Show the 2 sites you built (business site + store)
- Screenshots + what you did
- Host them for free (wordpress.com or 000webhost)

**Do:** Create a simple portfolio page with links to your work.

### Day 2: Freelance platforms
- Upwork: create profile, list services
- Fiverr: create gigs ("I will build a WordPress website")
- Freelancer: bid on projects

**Do:** Create profiles on Upwork and Fiverr. Write your descriptions.

### Day 3: Local businesses
- Walk into local businesses (restaurants, salons, gyms)
- "Your website looks outdated. I can build you a new one for $500."
- Most small businesses need websites. Many have bad ones.

**Do:** Visit 3 local businesses. Take note of their websites. Write down what's bad.

### Day 4: Cold outreach
- Email/visit 10 local businesses
- Offer: free website audit → they see what's wrong → you offer to fix it
- Price: $500-1500 per site

**Do:** Send 10 cold emails. "I noticed your website could be improved. Here are 3 things..."

### Day 5: Pricing
- Basic site (5 pages): $500-800
- With blog: $800-1200
- E-commerce store: $1500-3000
- Monthly maintenance: $50-100/month (hosting + updates)

### Day 6-7: First client
**Do:** Follow up with everyone. Someone will say yes. Build their site. Deliver fast. Ask for testimonial.

## 🗓️ Week 4 — Scale up

### Ongoing:
- Keep reaching out to businesses

### After:
