# 🔥 PyTorch — Deep Learning Framework

40 lessons covering PyTorch from tensor basics to production deployment.

## Structure

4 phases, 10 lessons each.

| Phase | Focus | Lessons | Practice |
|-------|-------|---------|----------|
| 01 | Tensors & Autograd | 01–10 | ✅ |
| 02 | Neural Networks & Training | 11–20 | ✅ |
| 03 | Advanced Architectures | 21–30 | ✅ |
| 04 | Production & Deployment | 31–40 | ✅ |

## How to Use

```bash
# Run a lesson's code
python code/01-tensor-basics.py

# Read the corresponding lesson
lessons/01-tensor-basics.md

# Do the practice exercises
practice/phase01-exercises.md
```

## Topics

**Phase 01:** tensors, operations, random seeding, autograd, gradient descent, linear regression from scratch, nn.Linear, loss functions, optimizers, binary classification.

**Phase 02:** nn.Sequential, activation functions, Dataset & DataLoader, train/val/test split, MLP, save/load models, early stopping, LR scheduling, regularization (dropout, weight decay), multi-class classification.

**Phase 03:** batch normalization, CNNs, image classification, RNNs, LSTM/GRU, transfer learning, fine-tuning, custom loss functions, TensorBoard, CNN integration.

**Phase 04:** GPU training, mixed precision, data parallel, TorchScript, ONNX export, TorchServe, hyperparameter optimization, gradient clipping, weight initialization, full production pipeline.

Requires PyTorch (`pip install torch`). All code files run standalone.
