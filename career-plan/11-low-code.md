# 🧩 Low-Code / No-Code — Baby Style (3 weeks)

**For:** Zero coding experience. Want to build apps without learning programming.

**Time:** 2-3 hrs/day | **Goal:** Freelance No-Code Developer

**Pay:** $40-80/hr (freelance) | **Difficulty:** Easiest track

---

## What is this?

- Low-code = build apps with drag-and-drop + small logic
- No-code = build apps with ZERO code
- Tools: Bubble (web apps), Airtable (databases), Zapier (automation), Webflow (websites)
- Businesses need apps but can't afford $100k developers. You build them cheap and fast.

---

## 🗓️ Week 1 — Bubble (Build Web Apps)

### Day 1: What is Bubble?
- Build web apps visually. Drag and drop.
- Users can sign up, log in, create data, see dashboards
- No code. Just clicks and simple logic.

**Do:** Sign up for Bubble (free plan). Go through the tutorial.

### Day 2: Bubble editor tour
- Design tab = drag elements onto page
- Workflow tab = what happens when user clicks something
- Data tab = database tables

**Do:** Create a page with a text input + button. When clicked, show what was typed.

### Day 3: Data in Bubble
- Create a "Thing" in the database (like an Object in code)
- Add fields (text, number, date, file)

**Do:** Create a "Task" data type with title + done/not done.

### Day 4: CRUD in Bubble
- Create: form → save to database
- Read: show list of items from database
- Update: mark task as done
- Delete: remove task

**Do:** Build a to-do app: add task, see list, mark done, delete.

### Day 5: User accounts
- Bubble has built-in signup/login
- Each user sees only their own data (privacy rules)

**Do:** Add signup + login to your to-do app. Each user has their own tasks.

### Day 6: Workflows + conditions
- "When button is clicked" → "If field is empty" → "Show error" → "Else save"

**Do:** Add validation: can't add empty task. Show error message.

### Day 7: Deploy
- Click "Deploy" in Bubble. Your app is live with a URL.
- Share it with anyone.

**Do:** Deploy your to-do app. Share with a friend.

## 🗓️ Week 2 — Airtable + Zapier

### Day 1: What is Airtable?
- Like Excel but more powerful. Database with a spreadsheet UI.
- Tables, fields, linked records, views (grid, calendar, gallery)

**Do:** Sign up for Airtable (free). Create a table for "Clients" (name, email, phone, notes).

### Day 2: Airtable views + filters
- Grid view = like Excel
- Calendar view = items with dates
- Gallery view = for images
- Filter: show only active clients

**Do:** Add 10 clients. Create a calendar view for meetings. Filter by status.

### Day 3: Airtable forms
- Share a form link. Anyone can submit. Data goes into your table.
- No coding. Built in.

**Do:** Create a "Contact Us" form. Submit test data. See it in the table.

### Day 4: What is Zapier?
- Connects apps together. "If X happens in App A, do Y in App B"
- Example: "If someone fills my form (Airtable) → send email (Gmail)"

**Do:** Create a Zap: new Airtable record → send Gmail notification.

### Day 5: Zapier workflows
- Triggers (new email, new row, new file) → Actions (create, update, notify)
- Filters: only run if condition is met

**Do:** Create a Zap: new Gmail with attachment → save to Google Drive → log in Airtable.

### Day 6: More Zaps
- Slack + Airtable: new Airtable record → post in Slack
- Typeform → Airtable: form responses go to database

**Do:** Create a Zap that sends you a Slack message when a new Airtable record is added.

### Day 7: Zapier project
- Build: "Customer feedback system"
- Typeform (collect feedback) → Airtable (store) → Slack (notify team) → Thank you email (Gmail)

## 🗓️ Week 3 — Webflow + Freelance

### Day 1: What is Webflow?
- Build professional websites visually (like WordPress but better)
- Drag and drop. No code. Clean code output.
- Used by designers and freelancers.

**Do:** Sign up for Webflow (free). Build a 1-page portfolio.

### Day 2: Webflow layout
- Sections → containers → divs → text/images
- Flexbox + grid (visual, no CSS needed)

**Do:** Build a landing page: hero, features, pricing, footer.

### Day 3: Webflow interactions
- Animations: fade in on scroll, hover effects, tabs
- No code. Just click and set triggers.

**Do:** Add an animation: header fades in when page loads.

### Day 4: Webflow CMS
- CMS = database for blog posts, projects, team members
- Create collection → add fields → design template → publish

**Do:** Create a blog collection. Write 3 posts. Show them on a blog page.

### Day 5: Portfolio + Pricing
- Your freelance site:
  - "I build web apps and websites. No coding needed. Fast. Affordable."

**Do:** Build your freelance website with Webflow: services, portfolio, contact.

### Day 6: Freelance platforms
- Upwork: "No-Code Developer", "Bubble Developer", "Webflow Designer"
- Fiverr: gigs starting at $200
- Local businesses: $500-2000 for a website or simple app

**Do:** Create Upwork + Fiverr profiles. List 3 services.

### Day 7: First client
- Offer to build a simple app for a local business for $500
- Examples: booking system, inventory tracker, customer directory
- Deliver fast. Ask for testimonial. Get referrals.

> ✅ **Low-code = fastest way to build things people will pay for.** No programming required. After you get clients, you can learn real code to charge more. Many successful freelancers started with no-code then added coding later.
