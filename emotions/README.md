## The Framework

Every emotion doc in this folder follows the same structure:

```
1. Evolutionary origin (why this emotion exists)
2. Neurochemistry (what chemicals drive it)
3. Brain areas (where it lives in the brain)
4. Adaptive function (what it was supposed to do)
5. Triggers (what sets it off)
6. Modern mismatch (why it fucks you up now)
7. Differential diagnosis (how it differs from similar emotions)
8. Practical protocol (how to work with it)
```

## The Core Insight (Read This First)

> **Every emotion is a neurochemical program that evolved to solve a specific survival problem. None of them are "bad" or "good." They are all ADAPTIVE in the right context and MALADAPTIVE in the wrong context. The goal is not to eliminate "negative" emotions. The goal is to understand what each emotion is trying to do for you, thank it for its service, and choose whether its ancient solution applies to your modern situation. Every protocol in these docs is designed to help you do exactly that.**
